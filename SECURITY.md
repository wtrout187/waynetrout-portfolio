# Security Policy

## Supported Versions

This is a static portfolio site. Only the `main` branch is maintained.

## Reporting a Vulnerability

- Email: [wayne.trout@ai-leader.com](mailto:wayne.trout@ai-leader.com)
- GitHub: open a private security advisory or contact via LinkedIn DM.

Please include:
- A clear description of the issue and impact
- Steps to reproduce (URLs, payloads, screenshots)
- Suggested remediation if known

## Response & Remediation Targets

- Acknowledge receipt: 2 business days
- Initial assessment: 5 business days
- Fix or mitigation: 15 business days for critical/high; 30 days for medium/low

## Scope / Out of Scope

- In scope: XSS, content injection, mixed-content issues, auth/session weaknesses if introduced later, leaks of secrets/tokens.
- Out of scope: DoS rate limits for this static site, issues requiring control of GitHub account, findings without a working proof of concept.

## Coordinated Disclosure

Please allow us to validate and fix before public disclosure. We will credit researchers upon request.
